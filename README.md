# Groceries Order Page
#### The page contains a Form that prompts user to enter 3 types of groceries of choice. 1-05-2018.
#### By **Ephantus Karanja**
## Description
This form returns the types of groceries the user inputs and orders them in unordered list.
## Setup/Installation Requirements
* The page can be opened using browsers such as mozilla firefox and google chrome for optimum views and results.

## Known Bugs

## Technologies Used
The page has been developed using html,javascript and jquery. Bootstrap and css were used to style the page.
